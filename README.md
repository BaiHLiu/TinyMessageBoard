## Usage
### 需要PHP >=7.0(以及PDO支持)，Mysql >=5.6，python >=3.6（可选）
新建数据库，导入./bbs.sql文件，修改./inc/conn_default.php为./inc/conn.php，并修改其中数据库相关字段。
另外可以使用./misc/getYiyan_default.py为数据库增加若干测试数据。
